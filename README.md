## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {linat, ".*",
      {git, "git@github.com:sdytr/linat.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
